Дії для використання програмного продукту:
Алгоритм дій користувача: Підходить у випадку, коли у користувача є персональний комп’ютер з ОС Windows.
1.	Завантажити виконуваний файл main.exe.
2.	Розмістити файл у будь-якій зручній директорії.
3.	Запустити файл подвійним натисканням лівої кнопки миші.
Програма не залишає після себе жодних файлів, тож нічого видаляти не потрібно

Алгоритм дій програміста (для продовження розробки): Для розгортання середовища розробки та внесення змін у вихідний код необхідно:
1.	Встановити компілятор C++ (MinGW або MSVC) та систему збірки CMake.
2.	Встановити інтерпретатор Python версії 3.10 або новішої.
3.	Встановити необхідні бібліотеки Python командою: pip install tk pyinstaller.
4.	Відкрити проект у середовищі (наприклад CLion) 
5.	Скомпілювати динамічну бібліотеку Solitaire.dll з прапорцем статичної лінковки. 
6.	Запустити клієнтську частину через main.py або зібрати новий .exe файл за допомогою PyInstaller.
